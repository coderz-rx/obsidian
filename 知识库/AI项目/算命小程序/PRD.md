# 算命小程序 PRD

> **产品定位**：以"算命/命理"为载体的情绪价值小程序，对标性格分类类产品，用轻量互动+AI生成内容满足用户的好奇心与情感需求
>
> **目标上线**：2026年5月

---

## 一、产品概述

### 1.0 核心问题：为什么不用 DeepSeek 直接算？

这是最关键的问题。如果用户能直接打开 DeepSeek 输入"帮我算八字"，为什么要用你的小程序？

**直接给 DeepSeek vs 用小程序的区别**：

| 维度 | 直接问 DeepSeek | 用算命小程序 |
|-----|----------------|-------------|
| **输入体验** | 手动输入出生日期、排盘、解释需求 | **一键选择**，结构化输入，3秒完成 |
| **结果呈现** | 大段文字，需要自己整理 | **精美卡片**，分模块展示（事业/感情/财运），**可直接分享朋友圈** |
| **场景深度** | 通用对话，需要反复追问 | **预置场景**：每日运势、缘分匹配、流年分析、AI问答，开箱即用 |
| **Prompt 质量** | 普通人不会写优质 Prompt | **隐藏 Prompt 工程**：角色设定+古籍引用+正向引导+钩子设计，专业级输出 |
| **社交传播** | 复制粘贴，体验差 | **原生分享卡片**，带二维码回流，病毒式传播 |
| **持续价值** | 每次重新输入，无记录 | **历史档案**：保存每次测算，追踪运势变化，建立用户粘性 |
| **情绪仪式感** | 聊天界面，像问问题 | **氛围感设计**：排盘动画、求签动效、古风界面，强化"被命运眷顾"的情绪体验 |

**一句话总结**：DeepSeek 是**原材料**，小程序是**成品+品牌+体验+社交闭环**。用户买的不是 AI 能力，是**方便、好看、可分享的情绪满足**。

---

### 1.1 核心洞察
- **对标产品**：性格分类类小程序（数百万用户）
- **底层逻辑**：用户不是在求"准"，是在求"被理解"和"确定性安慰"
- **情绪价值 > 功能价值**：算命比性格测试更有"神秘感"和"传播性"

### 1.2 产品一句话
> 不是卖 AI 算命能力，而是卖**30秒的仪式感+可分享的情绪价值**——让用户零门槛获得精美、正向、可发朋友圈的命理解读。

### 1.3 核心壁垒（为什么不会被 DeepSeek 替代）
1. **体验壁垒**：输入便捷性 + 视觉设计 + 分享动线，AI 模型本身不解决 UX
2. **场景壁垒**：每日运势推送、缘分匹配、历史档案——时间维度上的粘性
3. **社交壁垒**：朋友圈传播、情侣互测、好友对比——网络效应
4. **内容壁垒**：Prompt 工程 + 古籍知识库 + 正向话术体系，普通人不会配
5. **数据壁垒**：用户八字档案 + 测算历史 → 个性化越用越准（长期）
6. **文化壁垒**：周易八卦的专业起卦、卦象推演、爻辞引用——DeepSeek 不会主动用卦象框架回答问题，这是结构化产品设计才能提供的

### 1.4 目标用户
| 用户画像 | 特征 | 使用场景 |
|---------|------|---------|
| 年轻职场人 | 22-30岁，工作焦虑，未来迷茫 | 午休/睡前刷到，好奇测试 |
| 情感困惑者 | 恋爱/婚姻决策期 | 感情不顺时寻求"天意"指引 |
| 社交分享者 | 喜欢发朋友圈/群聊 | 测完觉得准，转发给朋友 |

## 二、核心功能模块

### 2.1 MVP 功能（五月上线版）

#### 功能1：八字命理 + 先天八卦盘
- **输入**：出生日期（公历/农历切换）+ 出生时间（可选）
- **输出**：
  - 八字排盘（四柱）
  - **先天八卦命盘**：根据八字五行属性映射到八卦方位，生成个人卦象图
  - AI生成的人格特质解读（500字以内）
- **亮点**：
  - 用"古籍引述"增加权威感（如《滴天髓》云...、"《易经·系辞》有言..."）
  - **八卦可视化**：太极八卦图+个人命卦高亮，仪式感拉满
  - 正向为主，负面委婉包装（"先苦后甜"、"大器晚成"）
- **小白说明**：
  > **八字是什么？** 就是你出生年月日时的天干地支，共八个字，传统命理认为这八个字藏着你的性格密码。  
  > **命卦是什么？** 八卦有八个符号（乾☰、坤☷、震☳、巽☴、坎☵、离☲、艮☶、兑☱），根据你的生辰推算出属于你的那一个，就像星座里的"你是狮子座"。  
  > **看不懂八卦图？** 没关系，我们会标出你的专属卦象，配上白话解读，告诉你这意味着什么性格、什么运势趋势。

#### 功能2：每日运势（卦象版）
- **每日生成**：今日运势评分 + 3条指引（事业/感情/财运）
- **每日卦象**：根据日期推演出今日之卦（如"今日为 乾卦·九五，飞龙在天"）
- **互动设计**：
  - 可"求签"——随机抽取一条签文
  - 可"摇卦"——模拟铜钱摇卦，生成六爻卦象
- **留存钩子**：订阅推送（每日8点"今日卦象已更新"）
- **小白说明**：
  > **今日之卦是什么？** 每天根据日期算出一个《易经》里的卦象，就像"今日幸运色是红色"一样简单。比如"乾卦·九五"=今天适合主动出击，有大展身手的机会。  
  > **求签是什么？** 就像去庙里抽签一样，心里默念一个问题，随机抽一条签文给你指引。比直接看运势更有仪式感。  
  > **摇卦是什么？** 传统算命师傅用三枚铜钱摇六次得到卦象，我们把它做成了动画版，点一下就能"摇"出结果，不用懂原理也能玩。

#### 功能3：缘分匹配（八卦合婚）
- **输入**：双方出生信息
- **输出**：
  - 八字合婚分析
  - **八卦合婚**：双方命卦的八卦方位关系（如"乾男配坤女，天地交泰"）
  - 缘分指数 + 相处建议
- **传播点**：情侣测完分享到朋友圈"我们的卦象相合度95%，乾男坤女，天地交泰"
- **小白说明**：
  > **八字合婚是什么？** 把两个人的生辰八字放在一起看，传统说法里能看出性格互补还是相冲。就像星座配对里的"天蝎座配双鱼座"。  
  > **八卦合婚是什么？** 两个人的命卦放在一起，看是"相生"还是"相克"。比如乾（天）配坤（地）=天地交泰，寓意天作之合；坎（水）配离（火）=水火既济，寓意相互调和。我们会用大白话告诉你"你们这对是互补型还是相似型"、"相处时要注意什么"。  
  > **看不懂卦名？** 每个卦名下面都有 emoji 和一句话翻译，比如"乾卦=天，代表领导力、上进心"，一看就懂。

#### 功能4：AI 周易问答（卦象式回答）
- **形式**：用户提问（"我今年能升职吗？""TA对我有意思吗？"）
- **输出**：
  - AI结合命理框架给出回答
  - **自动起卦**：根据问题性质和提问时间，推演一个卦象作为"天意参考"
  - 引用《易经》卦辞、爻辞作为回答框架（如"《易经·乾卦》九五：飞龙在天，利见大人..."）
- **限制**：免费3次/天，超限引导分享/看广告解锁
- **小白说明**：
  > **什么是起卦？** 就是你问一个问题，系统帮你"问天"得到一个卦象作为参考。就像抛硬币做决定，但比硬币更有文化感。  
  > **卦辞爻辞是什么？** 《易经》里的话，古人用来解释每个卦象的含义。我们会给出原文+白话翻译，比如"飞龙在天，利见大人"=你现在正处于上升期，适合主动展示自己，会遇到贵人。  
  > **这玩意准吗？** 我们的定位是"给你一个新的思考角度"，不是替你决定。就像朋友给你的建议，信则有，不信则无，重要的是帮你理清思路。

#### 功能5：六爻占卜（新增）
- **形式**：用户心中默念问题 → 点击"摇卦" → 模拟三枚铜钱摇六次
- **输出**：
  - 生成六爻卦象（本卦 + 变卦）
  - AI结合卦象解读问题（事业/感情/财运/决策）
  - 引用《易经》卦辞、爻辞原文
- **仪式感**：铜钱声音动效 + 八卦背景动画
- **收费点**：每日免费1次，额外次数需分享/付费
- **小白说明**：
  > **什么是六爻？** 就是摇六次铜钱，每次记录正反面，最后得到六条线（要么断开要么连上），这六条线组成一个"卦"。六条线的变化就像"六个开关的不同组合"，一共有64种可能。  
  > **本卦和变卦是什么？** 本卦=你现在的情况；变卦=事情发展的趋势。就像"现状"和"未来"两个视角。我们两张图都会展示，告诉你"现在该做什么"以及"以后会怎样"。  
  > **怎么摇？** 不用真铜钱！点"开始摇卦"，看动画里的三枚铜钱旋转落下，记录正反面。连续点六次就完成。整个过程就像抽奖，比直接看文字有意思多了。  
  > **需要懂易经吗？** 完全不需要。我们会把卦象翻译成"人话"，比如"这个卦象说现在不宜轻举妄动，宜守不宜攻"，直接给行动建议。

#### 功能6：周易经典（内容板块）
- **六十四卦速查**：每卦配有原文+白话解读+应用场景
- **每日一句易经**：随机推送《易经》经典语句
- **八卦入门**：小白友好的周易知识科普
- **小白说明**：
  > **六十四卦是什么？** 《易经》里的64个"人生场景模板"，每个卦描述一种处境和应对方法。比如"乾卦"讲如何做一个强者，"坤卦"讲如何做一个好的支持者，"屯卦"讲创业初期的困难。我们把它做成了"人生问题搜索引擎"，输入你的困惑就能找到对应的卦。  
  > **每日一句是干嘛的？** 每天送你一句《易经》里的智慧，配上白话解释。比如"天行健，君子以自强不息"=像天一样不断运动，强者要永远自我鞭策。适合发朋友圈装文化人。  
  > **八卦入门给谁看？** 如果你连"八卦是不是娱乐新闻"都分不清，点这个就对了。用最通俗的语言讲：八卦是什么、为什么有八个、每个代表什么性格、怎么用它理解人际关系。看完你就能跟朋友吹牛"我是坤卦人格，你是乾卦人格，咱俩天地配"了。

## 三、用户体验流程

### 3.1 新用户路径
```
小程序打开 → 3秒动效开场（太极八卦旋转 + "天机泄露"氛围） → 选择测算类型

```

## 四、商业模式

### 4.1 收入设计
| 层级 | 功能 | 价格 |
|-----|------|------|
| 免费 | 基础八字解读、每日运势摘要 | 0元 |
| 分享解锁 | 完整版报告、额外问答次数 | 分享到1个群/朋友圈 |
| 会员（月卡） | 无限问答、手相识别、专属大师 | 9.9-19.9元/月 |
| 单次付费 | 详细流年运势、合婚报告 | 6.9-29.9元/次 |

### 4.2 成本结构
- **AI调用**：**DeepSeek API（主力）** + Kimi API（补充），成本可控
- **服务器**：微信小程序云开发（初期低成本）
- **推广**：朋友圈广告/小红书种草（自然传播为主）

### 4.3 API 成本对比（关键）

| 模型 | 输入价格 | 输出价格 | 性价比评价 |
|-----|---------|---------|-----------|
| **DeepSeek-V3** | **¥1 / 百万 tokens** | **¥2 / 百万 tokens** | 🏆 **极度便宜，首选** |
| **DeepSeek-R1** | ¥4 / 百万 tokens | ¥16 / 百万 tokens | 推理能力强，适合复杂分析 |
| Kimi (8K) | ¥12 / 百万 tokens | ¥60 / 百万 tokens | 中文好，但贵 **10-30倍** |
| Kimi (128K) | ¥60 / 百万 tokens | ¥60 / 百万 tokens | 长文本场景用 |

**测算示例**：一次命理解读约消耗 800-1500 tokens
- DeepSeek 成本：约 **¥0.002~0.003**（几乎可以忽略）
- Kimi 成本：约 **¥0.05~0.1**（贵 20-50 倍）

**结论**：DeepSeek 便宜到离谱，建议作为主力。Kimi 可以在需要高质量长文本时偶尔调用。

## 五、技术方案

### 5.1 技术栈建议
| 层级 | 技术选型 |
|-----|---------|
| 前端 | 微信小程序原生 / Taro |
| 后端 | 微信云开发（云函数+云数据库）|
| AI生成 | **DeepSeek API**（性价比首选）/ Kimi API（中文质量优）|
| 排盘算法 | 开源八字排盘库（如 python-bazi）或自研 |
| 支付 | 微信支付 |

### 5.2 AI Prompt 策略
- **系统Prompt**：设定角色为"精通子平八字、紫微斗数、周易六爻、梅花易数的命理师"
- **生成约束**：
  - 70%正向内容 + 30%委婉提醒
  - **引用古籍增加可信度**：《滴天髓》、《三命通会》、《易经·系辞》、《易经》卦辞爻辞
  - **八卦框架**：回答时融入卦象概念（如"您命卦为坎卦，坎为水，水曰润下..."）
  - 避免具体医疗/法律建议（合规）
  - 每段结尾留钩子（"完整流年分析需解锁..."）

### 5.3 八卦命卦推演逻辑
- **先天八卦**：根据出生年份计算命卦（男命：(100-出生年后两位)/9取余；女命：(出生年后两位-4)/9取余）
- **后天八卦**：结合八字五行属性映射八卦方位（金-乾兑、木-震巽、水-坎、火-离、土-坤艮）
- **卦象展示**：生成可视化八卦图，命卦高亮显示，配以卦辞解读

### 5.4 视觉设计与氛围营造
- **主视觉**：太极八卦图为核心元素，贯穿所有页面
- **色彩体系**：朱砂红 + 玄黑色 + 金色点缀，传统又高级
- **动效设计**：
  - 起卦时：铜钱旋转落卦动画 + 八卦方位依次点亮
  - 排盘时：太极图旋转，八卦符号逐个浮现
  - 结果页：命卦高亮 + 太极背景微动效
- **字体选择**：标题用书法体（如站酷庆科黄油体），正文用清晰宋体
- **背景质感**：宣纸纹理 + 水墨晕染效果
- **声音设计**：铜钱碰撞声、古琴背景音乐（可选开关）

### 5.5 合规注意
- ⚠️ **关键**：在显著位置标注"本内容仅供娱乐参考，不构成任何决策建议"
- 避免涉及：医疗诊断、投资建议、赌博预测
- 用户协议中明确"AI生成内容，非真人命理师判断"

## 八、风险与对策

| 风险 | 对策 |
|-----|------|
| 微信审核不通过（迷信内容） | 强调"娱乐参考"定位，弱化"算命"用词，用"命理文化"、"性格分析"替代 |
