# 投资汇总

> 投资体系的核心脉络：道（理念）→ 法（分析）→ 术（交易）→ 戒（纪律）。以下内容按此结构凝练，只保留骨架和关键判断逻辑。

---

## 一、投资哲学（道）

### 1.1 收益不可能三角
高收益、短时间、低风险三者不可兼得。价值投资的取舍是：**高收益 + 长时间 + 低风险**。

### 1.2 两条路线

| 路线 | 核心逻辑 | 适用场景 | 关键能力 |
|------|---------|---------|---------|
| **价值投资** | 公司内在价值驱动，无视短期情绪 | 中长期、大资金 | 商业分析、财报解读、估值判断 |
| **短线交易** | 资金情绪驱动，跟随市场热点 | 短期、小资金 | 情绪感知、技术信号、纪律执行 |

> 价值投资吃逻辑，短线交易吃信号。两套体系不要混用，尤其忌讳：低位做短线，高位想转长线。

### 1.3 核心心态
- **耐心 > 一切** — 好买点等不到就换标的，强行上车必卖飞或深套
- **收益曲线目标**：向上 → 横住 → 再向上。好做时参与，难做时离场守收益
- **认知边界**：定好的预期到了就走，认知外的钱不赚
- **一致性**：要信早信，不信就始终套利；不要前期质疑、后期深信不疑

## 二、分析框架（法）

### 2.1 公司评估：四维度 + 五步法

**四维度（判断标的质量）：**
- 成长系数（空间）→ 盈利质量系数（质地）→ 估值风险系数（价格）→ 市场情绪系数（热度）
- 前三者向好 + 热度低 = 潜在机会；前三者向好 + 热度高 = 风险

**五步法（快速排雷）：**
1. 行业市场规模（天花板、周期性）
2. 公司市占率（是否上升、可替代性）
3. 毛利率+净利率（是否有话语权）
4. 公司调性（保守守业 vs 积极进取）
5. 估值合理性（PE/PB/PS/PEG综合判断）

**关键指标：**
- PEG < 1 合理，< 0.8 严重低估
- PE是静态回本，真实回本要看增速
- 财报三看：利润 → 资产负债 → 现金流量

### 2.2 深度分析：五大维度

```
行业维度（势）→ 市场地位（位）→ 盈利质量（质）→ 公司发展（速）→ 估值政策（价）
```

最终落点：**性价比定性** — 优势是否长期主导，风险是否短期可控。

### 2.3 短线选股：三层筛选

**资金面 → 消息面 → 技术面**，顺序不可颠倒。

**核心条件：**
- 板块出现连板梯队（4321/421成型），说明板块发酵，向上概率大
- 近20-30日有过涨停（有人气、主力有做多意图）
- 资金持续流入、无减持、图形好
- 龙头优先，杂毛不买（板块发酵后买非龙头=进去结账）

## 三、交易策略（术）

### 3.1 买入原则

**四大准则：**
1. 无量突破不参与
2. V型反转谨慎，多关注多次探底
3. 突破不直接追，等二次确认弱转强
4. 量价关系：缩量下跌还会跌，急跌放量会反弹，缩量上涨还会涨，急涨放量要回调

**两种节奏：**
- 等回踩均线（承受买不到风险）
- 拿得住（承受暂时亏损风险）
- 不能两全，根据市场情绪、成交量决策

### 3.2 止损与卖出

**核心逻辑：** 买入凭据发生变化，出现卖出信号，才卖。

**关键支撑/止损：**
- M20为重要支撑，跌破离场（尾盘为准）
- 盘中亏损超5%且跌破，也止损
- 半小时拉不回，果断走

**卖出场景提炼：**
- 该强不强就是弱（前一日高潮，次日不反弹）
- 该弱不弱就是强（前一日调整，次日逆势抗跌）
- 爆量后分歧无法转一致 → 走
- 连续上涨后爆量T字 → 走
- 市场极度热时，不出现连续亏钱效应不离场（拿住利润垫）

### 3.3 技术形态分类

**核心几类，不逐条罗列：**

| 类型 | 本质 | 代表形态 |
|------|------|---------|
| **反包类** | 空头释放后多头夺回主动权 | 反包大阴棒、看涨孕线、阳线反包 |
| **N字/龙回头** | 上涨中继，洗盘后二次启动 | 涨停N字、龙回头空中加油 |
| **放量突破** | 资金进场，确认趋势 | 放量破位红三兵、三点定势、首板破前高 |
| **缩量止跌** | 抛压衰竭，等待反转 | 单阳不破、缩量红中、仙人指路 |
| **趋势回踩** | 上升趋势中的低吸点 | 回踩M20、均线共振 |

### 3.4 做T与仓位

- 专做熟悉票，不用每天选股
- 持股1-2支，其他板块买ETF，**不要开超市**
- 小资金不平铺，每个方向最多一只，同时最多关注两个方向

## 四、风控纪律（戒）

### 4.1 绝对红线

- 不要因为害怕踏空乱买（钱赚不完，亏得完）
- 不要买减持股、杂毛、高位三浪五浪
- 不要开盘卖在均线之上且无掉头趋势的个股
- 不要无逻辑死拿，无逻辑卖出后又高看一眼
- 不要低位短线到高位切长线
- 不要随意更改交易规则（规则在冷静时定，定了就执行）

### 4.2 危险信号（立即降低仓位或离场）

- 主力资金持续大量流出
- 总体一直赚突然不赚甚至亏钱（环境已不适合当前操作水平）
- 板块资金持续流出 + 龙头破位（行情可能结束）
- 个股次日跌破M20且十点半前拉不回
- 爆量上涨后分歧无法转一致

### 4.3 每日复盘要点

- 当晚：热门板块、资金流入 → 决定持仓和次日计划
- 竞价：板块热点、涨停数是否符合预期 → 调整决策
- 盘后：检查是否遵守规则，优化体系，不盘中改规则

---

## 五、一句话核心

**价值投资**：找"好公司+好价格+拿得住"，赚企业成长的钱。
**短线交易**：找"热点板块+资金流入+技术信号+严格止损"，赚情绪波动的钱，纪律高于判断。

两者共通：**买入有理由，卖出有信号，耐心等机会，果断守纪律。**
