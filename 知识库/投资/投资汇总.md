# 投资汇总

> 投资体系的核心脉络：道（理念）→ 法（分析）→ 术（交易）→ 戒（纪律）。以下内容按此结构凝练，只保留骨架和关键判断逻辑。

---

## 一、投资哲学（道）

### 1.1 收益不可能三角
高收益、短时间、低风险三者不可兼得。价值投资的取舍是：**高收益 + 长时间 + 低风险**。

### 1.2 两条路线

| 路线 | 核心逻辑 | 适用场景 | 关键能力 |
|------|---------|---------|---------|
| **价值投资** | 公司内在价值驱动，无视短期情绪 | 中长期、大资金 | 商业分析、财报解读、估值判断 |
| **短线交易** | 资金情绪驱动，跟随市场热点 | 短期、小资金 | 情绪感知、技术信号、纪律执行 |

> 价值投资吃逻辑，短线交易吃信号。两套体系不要混用，尤其忌讳：低位做短线，高位想转长线。

### 1.3 核心心态
- **耐心 > 一切** — 好买点等不到就换标的，强行上车必卖飞或深套
- **收益曲线目标**：向上 → 横住 → 再向上。好做时参与，难做时离场守收益
- **认知边界**：定好的预期到了就走，认知外的钱不赚
- **一致性**：要信早信，不信就始终套利；不要前期质疑、后期深信不疑

## 二、分析框架（法）

### 2.1 公司评估：四维度 + 五步法

**四维度（判断标的质量）：**
- 成长系数（空间）→ 盈利质量系数（质地）→ 估值风险系数（价格）→ 市场情绪系数（热度）
- 前三者向好 + 热度低 = 潜在机会；前三者向好 + 热度高 = 风险

**五步法（快速排雷）：**
1. 行业市场规模（天花板、周期性）
2. 公司市占率（是否上升、可替代性）
3. 毛利率+净利率（是否有话语权）
4. 公司调性（保守守业 vs 积极进取）
5. 估值合理性（PE/PB/PS/PEG综合判断）

**关键指标：**
- PEG < 1 合理，< 0.8 严重低估
- PE是静态回本，真实回本要看增速
- 财报三看：利润 → 资产负债 → 现金流量

### 2.2 深度分析：五大维度

```
行业维度（势）→ 市场地位（位）→ 盈利质量（质）→ 公司发展（速）→ 估值政策（价）
```

最终落点：**性价比定性** — 优势是否长期主导，风险是否短期可控。

### 2.3 短线选股：三层筛选

**资金面 → 消息面 → 技术面**，顺序不可颠倒。

**核心条件：**
- 板块出现连板梯队（4321/421成型），说明板块发酵，向上概率大
- 近20-30日有过涨停（有人气、主力有做多意图）
- 资金持续流入、无减持、图形好
- 龙头优先，杂毛不买（板块发酵后买非龙头=进去结账）

## 三、交易策略（术）

### 3.1 买入原则

**四大准则：**
1. 无量突破不参与
2. V型反转谨慎，多关注多次探底
3. 突破不直接追，等二次确认弱转强
4. 量价关系：缩量下跌还会跌，急跌放量会反弹，缩量上涨还会涨，急涨放量要回调

**两种节奏：**
- 等回踩均线（承受买不到风险）
- 拿得住（承受暂时亏损风险）
- 不能两全，根据市场情绪、成交量决策

### 3.2 止损与卖出

**核心逻辑：** 买入凭据发生变化，出现卖出信号，才卖。

**关键支撑/止损：**
- M20为重要支撑，跌破离场（尾盘为准）
- 盘中亏损超5%且跌破，也止损
- 半小时拉不回，果断走

**卖出场景提炼：**
- 该强不强就是弱（前一日高潮，次日不反弹）
- 该弱不弱就是强（前一日调整，次日逆势抗跌）
- 爆量后分歧无法转一致 → 走
- 连续上涨后爆量T字 → 走
- 市场极度热时，不出现连续亏钱效应不离场（拿住利润垫）

### 3.3 技术形态分类

**核心几类，不逐条罗列：**

| 类型 | 本质 | 代表形态 |
|------|------|---------|
| **反包类** | 空头释放后多头夺回主动权 | 反包大阴棒、看涨孕线、阳线反包 |
| **N字/龙回头** | 上涨中继，洗盘后二次启动 | 涨停N字、龙回头空中加油 |
| **放量突破** | 资金进场，确认趋势 | 放量破位红三兵、三点定势、首板破前高 |
| **缩量止跌** | 抛压衰竭，等待反转 | 单阳不破、缩量红中、仙人指路 |
| **趋势回踩** | 上升趋势中的低吸点 | 回踩M20、均线共振 |

### 3.4 做T与仓位

- 专做熟悉票，不用每天选股
- 持股1-2支，其他板块买ETF，**不要开超市**
- 小资金不平铺，每个方向最多一只，同时最多关注两个方向

## 四、风控纪律（戒）

### 4.1 绝对红线

- 不要因为害怕踏空乱买（钱赚不完，亏得完）
- 不要买减持股、杂毛、高位三浪五浪
- 不要开盘卖在均线之上且无掉头趋势的个股
- 不要无逻辑死拿，无逻辑卖出后又高看一眼
- 不要低位短线到高位切长线
- 不要随意更改交易规则（规则在冷静时定，定了就执行）

### 4.2 危险信号（立即降低仓位或离场）

- 主力资金持续大量流出
- 总体一直赚突然不赚甚至亏钱（环境已不适合当前操作水平）
- 板块资金持续流出 + 龙头破位（行情可能结束）
- 个股次日跌破M20且十点半前拉不回
- 爆量上涨后分歧无法转一致

### 4.3 每日复盘要点

- 当晚：热门板块、资金流入 → 决定持仓和次日计划
- 竞价：板块热点、涨停数是否符合预期 → 调整决策
- 盘后：检查是否遵守规则，优化体系，不盘中改规则

---

## 五、一句话核心

**价值投资**：找"好公司+好价格+拿得住"，赚企业成长的钱。
**短线交易**：找"热点板块+资金流入+技术信号+严格止损"，赚情绪波动的钱，纪律高于判断。

两者共通：**买入有理由，卖出有信号，耐心等机会，果断守纪律。**

## 六、多因子买卖决策体系

> 以下体系基于投资目录全部笔记提炼，将买入/卖出条件拆分为可独立判断的因子，通过组合共振提高胜率，通过优先级避免信号冲突时的犹豫。

### 6.1 因子分类与定义

所有买卖决策由六类因子构成，按重要性排序：

```
市场环境（大势）→ 板块因子（方向）→ 风险因子（过滤）→ 个股因子（标的）→ 资金因子（确认）→ 技术因子（择时）
```

| 因子类别 | 核心作用 | 典型指标 |
|---------|---------|---------|
| **市场环境** | 决定是否参与 | 操作难度、情绪周期、连续亏钱效应 |
| **板块因子** | 决定交易方向 | 连板梯队、板块资金流入、竞价涨停数 |
| **风险因子** | 决定能否排除 | 高位/三浪五浪、杂毛、减持、极致高潮次日 |
| **个股因子** | 决定具体标的 | 涨停基因、位置、市值、无减持 |
| **资金因子** | 决定主力态度 | 净流入、筹码集中、量价配合 |
| **技术因子** | 决定具体买点 | 突破/回踩/形态/竞价信号 |

### 6.2 买入决策矩阵

**前提条件（必须全部满足，否则不交易）：**

| 前提 | 判断标准 | 不满足则 |
|------|---------|---------|
| **市场环境可参与** | 无连续亏钱效应，非极致退潮期 | 空仓观望 |
| **风险因子无触发** | 非高位三浪五浪、非杂毛、无减持、非极致高潮次日 | 排除该标的 |

**核心条件（至少满足3项，其中板块和个股至少各1项）：**

| 条件 | 满足标准 | 权重 |
|------|---------|------|
| **板块梯队成型** | 板块出现4321/421连板梯队，非单一龙头独舞 | 高 |
| **板块资金流入** | 板块资金持续流入，竞价涨停数符合预期 | 高 |
| **个股涨停基因** | 近20-30日有过涨停，说明主力有做多意图 | 中 |
| **个股位置合理** | 非已大幅发酵后的杂毛，流通盘不超100亿 | 中 |
| **资金确认** | 主力资金净流入，底部筹码集中 | 高 |
| **技术确认** | 放量突破/回踩均线不破位/30分时共振 | 中 |

**买入场景组合：**

| 场景 | 核心组合 | 辅助验证 |
|------|---------|---------|
| **突破买入** | 板块梯队成型 + 资金确认 | 放量突破压力位 + 个股涨停基因 |
| **回踩买入** | 板块资金持续流入 + 个股位置合理 | 回踩M20不破位 + 30分时共振 |
| **反包买入** | 板块未退潮 + 个股涨停基因 | 反包大阴线/看涨孕线成立 + 资金回流 |
| **低吸埋伏** | 市场环境可参与 + 风险无触发 | 前期有板+缩量震荡 + 资金未流出 |

**当日择时细节：**

| 竞价情况 | 处理方式 | 适用场景 |
|---------|---------|---------|
| 3%以内高开 | 直接介入 | 突破确认、强势延续 |
| 3%以上高开 | 等回踩分时均线 | 防追高、防高开低走 |
| 平开且9:40仍红 | 可进底仓 | 犹豫中走强，试探性参与 |
| 低开 | 不参与（等下一次放量突破） | 弱势信号，避免接盘 |
| 涨停开盘突然撤单+主力流出 | 不追，观察短期能否拉回 | 分歧信号，谨慎 |

### 6.3 卖出决策矩阵

**硬约束（触发即执行，不犹豫、不等待）：**

| 约束 | 触发条件 | 执行动作 | 例外情况 |
|------|---------|---------|---------|
| **亏损硬止损** | 持仓亏损超5% | 立即离场 | 无 |
| **技术硬止损** | 跌破M20且尾盘未收回 | 收盘前离场 | 盘中跌破但尾盘拉回，可观察 |
| **时间止损** | 买入后当日半小时内持续走弱，无反弹迹象 | 果断离场 | 无 |
| **大幅回撤** | 持仓个股单日大幅下跌超3%且不反弹 | 强制卖出 | 无 |

**软约束（需2项及以上同时触发，或单一触发+情绪确认）：**

| 约束 | 触发条件 | 配合确认 |
|------|---------|---------|
| **技术破位** | 高位股当日破五日线不收回 | 大盘不配合/板块退潮 |
| **量价背离** | 爆量上涨后次日分歧无法转一致 | 资金流出加剧 |
| **形态走坏** | 连续上涨后爆量T字、该强不强 | 板块龙头同步走弱 |
| **资金退潮** | 主力资金持续大量流出 | 板块资金同步流出 |
| **环境恶化** | 总体一直赚突然不赚甚至亏钱 | 连续2-3笔交易亏损 |
| **预期到达** | 定好的收益目标达到 | 无（主动止盈） |
| **逻辑变化** | 买入凭据发生变化（热点退潮、政策利空等） | 无（主动离场） |

**卖出场景组合：**

| 场景 | 触发因子 | 执行方式 |
|------|---------|---------|
| **止损离场** | 任一硬约束触发 | 立即执行，不抱侥幸心理 |
| **止盈离场** | 预期到达 + 技术出现该强不强 | 分批或一次性离场 |
| **保本离场** | 买入逻辑未兑现 + 环境恶化 | 停止加仓，找机会减亏离场 |
| **恐慌离场** | 环境极度恶化 + 多股同步破位 | 无条件降低仓位，离场观望 |
| **主动离场** | 逻辑变化 + 认知边界到达 | 不赚认知外的钱，果断走 |

### 6.4 优先级与冲突处理

**当多个信号冲突时的决策顺序：**

```
硬约束 > 预期管理 > 环境判断 > 资金信号 > 技术信号
```

**具体冲突示例：**

| 冲突场景 | 判断逻辑 | 最终决策 |
|---------|---------|---------|
| 个股技术走好但板块资金流出 | 板块资金 > 个股技术 | 不参与或减仓 |
| 个股破M20但尾盘拉回 | 尾盘确认优先于盘中波动 | 不触发硬止损 |
| 收益未到但环境极度恶化 | 环境 > 收益预期 | 离场守利润 |
| 技术形态完美但市场环境难做 | 市场环境 > 个股形态 | 空仓等机会 |
| 主力流入但板块梯队瓦解 | 板块梯队 > 个股资金 | 不参与 |

### 6.5 决策Checklist

**每日开盘前必查：**
- [ ] 市场环境是否可参与？（有无连续亏钱效应）
- [ ] 手中持仓是否触发硬约束？
- [ ] 今日计划交易标的的风险因子是否全部通过？
- [ ] 目标板块是否满足梯队/资金至少1项？
- [ ] 目标个股是否满足涨停基因/位置合理至少1项？

**每笔买入前必查：**
- [ ] 市场环境：可参与？
- [ ] 风险因子：无触发？
- [ ] 板块因子：至少1项满足？
- [ ] 个股因子：至少1项满足？
- [ ] 资金/技术因子：至少1项确认？
- [ ] 买入后止损位是否明确？（亏损%/技术位/时间）
- [ ] 买入后预期收益是否明确？（到达即走）

**每笔卖出前必查：**
- [ ] 是否触发硬约束？（是→立即执行）
- [ ] 是否触发2项及以上软约束？（是→执行）
- [ ] 是否到达预期收益？（是→主动止盈）
- [ ] 买入逻辑是否发生变化？（是→主动离场）
- [ ] 卖出的理由是否比买入时的理由更充分？
