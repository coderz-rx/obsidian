> 本文档汇总投资目录下所有笔记的核心内容，便于快速查阅和体系化回顾。

## 一、投资理念与核心原则

### 1.1 价值投资底层逻辑

**四维度评估公司：**
- **成长系数**：市场空间大小、未来增长空间
- **盈利质量系数**：收入质量、毛利率、增长持续性
- **估值风险系数**：当前估值是否合理
- **市场情绪系数**：市场关注度、资金热度

> 前三者向好+关注度高 = 风险很大；前三者向好+关注度低 = 潜在机会

**核心认知：**
- 价值投资可以无视仓位、情绪等干扰因素
- 想清楚想赚什么收益，不同收益伴随不同风险
- **高收益、长时间、低风险**是价值投资的追求（不可能三角的取舍）
- 垃圾股不敢上仓位，吃一口就跑容易赚十次亏一次
- **要信早信，不然就一直别信**

### 1.2 交易心态与纪律

**核心原则：**
- **卖出要果断，买入要慎重**
- **定好的预期到了就卖，认知外的钱不要赚**
- **股市赚钱不在于一次赚多少，持续小赚积累高额收益**
- **一年只需赚几次，把握全年仅有的几次好行情**
- **前后一致，知行合一** — 错了就认，对了到达预期也要卖
- **不要前期质疑、后期深信不疑** — 要信早信，不信就套利喝汤
- **钱赚不完，但是亏得完** — 没有把握不要乱买
- **不要因为害怕踏空乱买股票**
- **耐心是这个市场最宝贵的品质**
- **低位做短线，到高位企图切换长线** = 利润大幅回撤主因

**收益曲线目标：**
> 向上 → 横住 → 再向上。好做时参与赚钱，难做时离场守收益。

## 二、公司分析体系

### 2.1 快速评估（价值投资.md）

**五步法：**
1. **行业市场规模**：天花板多高？是否周期性？
2. **公司市占率**：是否逐年上升？可替代性高不高？
3. **毛利率+净利率**：是否有话语权？（警惕"科技壳、销售实"）
4. **公司调性**：保守守业 vs 积极进取（看增长、扩张、收购）
5. **估值合理性**：PE、PB、PS、PEG 综合判断

**关键指标：**
- **PE**：市值/扣非净利润，静态回本年限
- **PB**：市值/净资产，看家底
- **PS**：市值/营收，亏损股主要看这个
- **PEG**：PE÷净利润增速，**<1合理，<0.8严重低估**

> PE是静态回本，真实回本年限要看增速。例：PE=30、增速30%，实际回本约8.4年。

**财报三看：**
1. 看利润（收入、毛利、净利润、费用）
2. 看资产负债（现金、投资、并购、商誉）
3. 看现金流量（造血能力）

### 2.2 深度分析框架（价值投资分析模板.md）

**五大维度 + 综合定性：**

| 维度 | 核心问题 | 关键指标 |
|------|---------|---------|
| **行业** | 赛道天花板与可持续性 | 市场规模、CAGR、产业链位置、生命周期、周期属性 |
| **市场地位** | 是否龙头，有无护城河 | 市占率、竞争格局、技术/产能/客户/品牌壁垒 |
| **盈利质量** | 是否好生意，财务健康 | 毛利率、净利率、费用率、ROE、资产负债率、现金流 |
| **公司发展** | 保守还是进取，增长落地 | 增长趋势、战略聚焦度、产能扩张、研发投入、管理层 |
| **估值与政策** | 价格合理性与外部催化 | PE/PB/PS/PEG、估值分位、政策利好、市场情绪 |

**性价比定性：**
- **高性价比**：优势长期不可替代，风险短期可控，估值低估
- **中性性价比**：优势与风险并存，估值合理
- **低性价比**：优势不显著，长期风险>短期优势，估值高估

## 三、短线交易规则

### 3.1 选股逻辑

**核心方向：**
- **资金面 + 消息面 + 技术面** 三层筛选
- **板块热度 > 个股选择** — 板块出现连板梯队（4321/421成型）时重点关注
- 近20-30日有过涨停 + 在热点板块 + 图形好 + 资金流入 + 无减持
- **龙头要有带动性、抗跌性、主动性**
- **不要买杂毛**，板块发酵有涨幅后买杂毛=进去结账
- **不要买减持股**，高开超5个点不追
- **尽量不做高位票三浪五浪博弈**

**具体选股条件：**
- 近期有涨停（有人气、主力有做多想法）
- 突破后日线回踩M20不破位（30分时图找切入机会）
- 资金流入状态
- 结合第二天主力资金流入、消息面刺激决策
- 流通盘市值尽量不超过100亿（考虑套牢盘和拉升难度）

### 3.2 买入规则

**四大准则：**
1. **无量突破不参与**
2. **V型反转谨慎参与**，多关注多次探底个股
3. **突破不要直接追**，回头二次确认弱转强再上车
4. **缩量下跌还会跌，急跌放量会反弹，缩量上涨还会涨，急涨放量要回调**

**买入节奏：**
- **要么等回踩日级别均线**，承受买不到风险
- **要么拿得住**，承受暂时亏损风险
- 连续跌的买止跌阳线，**多等两个确认**
- 震荡上行的买阴线，趋势确认不要犹豫
- 前期有板，后面缩量震荡的，值得埋伏
- **不要开盘半小时内操作**，下午买第二天卖

**强势股次日策略：**
- 3%以内高开直接进，超过3%等回踩
- 平开：9:40还是红可以进底仓
- 低开：不参与（等再次放量突破本次高点）

### 3.3 止损与卖出规则

**核心止损原则：**
- **M20作为重要支撑线**，跌破离场（以尾盘为准）
- 盘中跌破且当天亏损超5%，也要止损
- **设置的止损位**，开盘半小时拉不回，或盘中半小时拉不回，则离场
- 强势股卖点等早盘十点后再决定，第二个30分钟K线抛压大果断走

**具体卖出场景：**
- 个股前一日高潮，次日高开回落反弹不到均线以上 → 卖出
- 前一日调整，次日继续低开，竞价无价格回落，但大幅下跌超3% → 强制卖出
- 前一日小涨/小跌，未跌破均线/极度一致情绪/长上影线/趋势走坏 → 坚定持股
- **开盘绝对不能卖在均线之上且未掉头向下的个股** — 要出现卖讯再卖
- 市场极度热时，不出现连续亏钱效应不离场，拿住优质股票
- 爆量上涨后第二天注意分歧是否转一致，转不了 → 离场
- 连续上涨后爆量T字 → 离场信号
- 涨停后次日放量红盘但有分歧、高位调整 → 警惕

**卖出铁律：**
- **买和卖都要有理由**
- 卖出时一定是买入凭据发生变化，出现卖出信号
- 买入时就想清楚：什么条件变了要卖？盈利多少卖？亏损多少卖？
- 主力资金持续大量流出 → 风险意识，及时降低风险偏好

### 3.4 技术形态买入法（买入机会、卖出风险.md）

| 形态 | 核心条件 | 止损点 |
|------|---------|--------|
| **反包放量突破大阴棒** | 首日大阳线(>5%)→次日放量大阴线→第三天反包成功 | 次日低点 |
| **涨停N字买入法** | 首板涨停→调整不破首板最低点→再涨停 | 第二个涨停最低价 |
| **放量破位红三兵** | 底部放量，连续3天阳线，最好突破压力位 | 第一根阳线底部 |
| **看涨孕线** | 大阴线→跳空高开小阳线被包住→大阳突破大阴高点 | 大阴线低点 |
| **龙回头空中加油** | 第一轮上涨后盘整换手>20%，跌破5/10日线，5%大阳回十日线上方 | 当天K线实体最低点 |
| **三点定势** | 加速赶底→小阳线反弹→倍量高开低走大阴线→后续放量突破大阴线高点 | 大阴线低点 |
| **仙人指路** | 长上影线试探压力，后续放量突破 | 仙人指路当天实体最低点 |
| **单阳不破** | 首板破平台，第二天尝试介入，持续缩量不跌破涨停范围，地量后放量大阳 | 涨停K线底部 |
| **旭日东升** | 持续阴跌后跳空阳线（大周期均线上的更好） | 跳空阳线低点 |

**做T策略：**
- 专做熟悉的票，不用每天选股
- 开盘急跌5个点+缩量+远离均线 → 买入，反弹到均线卖底仓
- 开盘十分钟急拉未封板 → 先卖底仓，跳水再买回
- 大盘涨但票不涨 → 先考虑卖底仓（个股可能跌更狠）
- 大盘跌但票不跌 → 先买进，等反弹卖出
- 低位震荡底部不断抬升 → 第三个低点买等量底仓，冲高卖

## 四、交易复盘与迭代

### 4.1 复盘核心原则（交易复盘.md）

- 定好的预期到了就卖，认知外的钱不要赚
- 资金持续净流入+底部筹码集中+有突破趋势 → 尝试买入，定好收益/亏损预期
- 市场即将走弱 → 不买股，别把自己当特殊
- **成功率高的才能出手**
- 没有逻辑闭环不要买，买了要拿住，逻辑变了才卖
- 高位股当日破五日线不收回 → 收盘卖；突发事件半小时不收回再卖
- 前期涨幅过高+热点下降 → 不要追高，关注当下热点
- 主力资金大幅度流出 → 提高警惕，至少不加仓
- 总体一直赚突然不赚甚至亏钱 → 停止买票，卖出缓一缓
- **平铺策略不适合小资金**，每次出手每个方向最多买一只，同时最多关注两个方向

### 4.2 每日复盘要点（短线交易规则.md）

- 当晚复盘：热门板块、资金流入情况 → 决定持仓和次日交易计划
- 竞价阶段：看板块热点、资金流入、涨停数是否符合预期 → 调整决策
- 事后复盘优化交易体系，**不要交易途中随意更改规则**
- 严格执行记录，遵守纪律

## 五、风险控制清单

### 5.1 高风险信号（立即警惕）

- [ ] 主力资金持续大量流出
- [ ] 持有的股票总体一直赚突然不赚甚至亏钱
- [ ] 市场处于极度热+连续亏钱效应出现
- [ ] 个股前一日高潮，次日高开回落不反弹到均线
- [ ] 爆量上涨后分歧无法转一致
- [ ] 连续上涨后爆量T字
- [ ] 买入后次日跌破M20且十点半前拉不回
- [ ] 板块资金持续流出+龙头破位
- [ ] 高位股三浪五浪博弈
- [ ] 缩量下跌未止跌

### 5.2 操作红线（不可触碰）

- [ ] **不要买杂毛**（板块发酵后买非龙头）
- [ ] **不要买减持股**
- [ ] **不要开盘卖在均线之上且无掉头趋势的个股**
- [ ] **不要因为害怕踏空乱买**
- [ ] **不要买已经走完三浪五浪的高位票**
- [ ] **不要平铺开超市**（持股1-2支，其他买ETF）
- [ ] **不要博弈分歧特别大的弱转强**
- [ ] **不要所有钱都赚**（难做时离场等机会）
- [ ] **不要低位短线到高位切长线**
- [ ] **不要无逻辑死拿，无逻辑卖出后又高看一眼**
